# NoFog Mod

A client-side mod for Minecraft Fabric that completely disables fog. Press F6 to toggle fog on/off.

## Features
- Complete fog removal when disabled
- Toggle fog with F6 key (configurable in Minecraft controls)
- Client-side only functionality
- Works with Minecraft 1.21.1

## Requirements
- Minecraft 1.21.1
- Fabric Loader 0.16.0+
- Java 21+
- Fabric API

## Building
```bash
./gradlew build
```

## Installation
1. Build the mod using Gradle
2. Copy the JAR file from `build/libs/` to your Minecraft `mods` folder
3. Launch Minecraft with Fabric
4. Press F6 to toggle fog on/off

## Controls
- **F6** - Toggle fog on/off (can be changed in Minecraft Controls settings under "NoFog" category)
